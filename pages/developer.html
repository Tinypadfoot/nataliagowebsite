I am a new website deveoper, learning on my own through online courses. In the last year, I learned how to code using python in the Django framework. While I love how Django is designed to simplify building complex dynamic websites, it is a little too complicated for creating a personal site. Deploying it is a whole other thing.

I built this site using Jekyll and github pages, using a theme I borrowed from Konstantinos CHorianopoulos on Github and customized for my needs. It was originally a personal card but I added pages and navigation to link them together to showcase all the things I am doing at the moment.

Building a site using Jekyll is a lot easier and the site loads faster once you test it on a browser compared to Django. There are components that I do miss from Django but for a simple personal website, I found that Jekyll is more suitable.
